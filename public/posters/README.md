# Lesson Poster Images

Optionale Standbild-Poster fuer die Video-Player.

## Namens-Konvention

```
public/posters/<moduleSlug>-<lessonSlug>.jpg
```

Beispiel:

```
public/posters/module6-6-3.jpg
```

## Wirkung

Wenn vorhanden, wird das Bild als `poster` des `<video>` verwendet.
Ohne Poster zeigt der Browser einen leeren schwarzen Frame, bis der
User Play drueckt.

## Empfehlung

- Format: JPG
- Aufloesung: 1280x720 (16:9) reicht meistens
- Groesse: unter 200 KB pro Datei
